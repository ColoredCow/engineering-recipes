# AI Onboarding Guide
> How to use the Onboarding Project and Skills on claude.ai

---

## What you need

Before starting, make sure you have:

1. A claude.ai account — sign in with your work email
2. Access to the shared Onboarding Project — link in `ONBOARDING.md`
3. `CLAUDE_BASICS.md` read — takes ~10 minutes

---

## Opening the Onboarding Project

### Step 1 — Go to Projects
Open claude.ai and click **Projects** in the left sidebar.

![Left sidebar with Projects highlighted](screenshots/01_projects_sidebar.png)

### Step 2 — Open the Onboarding Project
Find the project named **"New Member Onboarding"** and click it.

![Projects list with New Member Onboarding highlighted](screenshots/02_onboarding_project.png)

### Step 3 — Start your onboarding
In the chat, type exactly:

```
I'm a new member. Start my onboarding.
```

The agent will take it from there. The session takes 20–30 minutes.

![Chat window with trigger phrase typed](screenshots/03_trigger_phrase.png)

---

## Using a Skill

Skills are shared Claude Projects pre-configured for specific tasks. Each skill has its own Project.

### Step 1 — Pick the right skill Project
From the Projects list, select the skill you need. Each Project is named after its purpose.

![Projects list showing skill projects](screenshots/04_skill_projects.png)

### Step 2 — Just start chatting
Open the Project and describe your task. The skill instructions are already loaded — you don't need to paste anything.

![Skill project open with a user message and Claude response](screenshots/05_skill_in_use.png)

### Step 3 — Check the repo for available skills
New skills are added regularly. Check `/skills/` in the engineering recipes repo to see what's available.

> **No skill for your task?** That's a contribution opportunity — see `ONBOARDING.md` → How to contribute.

## Quick reference

| What | Where |
